Git錯誤提示之：fatal: Not a git repository (or any of the parent directories): .git

產生原因：一般是没有初始化git本地版本管理倉庫，所以無法執行git命令

解决方法：操作之前執行以下命令行:
git init
然后執行一下git status查看狀態信息，good，問題解決。
